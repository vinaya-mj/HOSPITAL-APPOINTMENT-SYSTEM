# 🏥 Hospital Appointment System

A simple Java-based Hospital Appointment Management System designed to demonstrate database connectivity and appointment handling concepts.

---

## 📌 Features

* Hospital database initialization
* Appointment management
* Java-based application structure
* Database connectivity support
* Easily extendable architecture

---

## 📂 Project Structure

```bash
Hospital-Appointment-System/
│
├── DB.java
├── HospitalApp.java
└── HospitalDBSetup.java
```

### File Description

| File                   | Description                                   |
| ---------------------- | --------------------------------------------- |
| `DB.java`              | Handles database connection and operations    |
| `HospitalApp.java`     | Main application entry point                  |
| `HospitalDBSetup.java` | Creates and initializes the hospital database |

---

## 🛠️ Technologies Used

* Java
* JDBC
* MySQL (or compatible relational database)

---

## 🚀 Getting Started

### Prerequisites

* Java JDK 8 or later
* MySQL Database
* JDBC Driver
* IDE such as IntelliJ IDEA, Eclipse, or VS Code

### Compile the Project

```bash
javac *.java
```

### Run the Application

```bash
java HospitalApp
```

---

## ⚙️ Database Setup

Execute the database initialization class:

```bash
java HospitalDBSetup
```

This will create the required tables and prepare the database for use.
